Browser extension to

**S**ave

**I**tems

**F**rom

**T**abs,

**Y**o!

Searches open tabs for media (images, audio, video), saves them, and closes the tabs.

Get it as a [Firefox Add-on](https://addons.mozilla.org/en-US/firefox/addon/sifty/).
